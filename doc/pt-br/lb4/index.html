LoopBack é a plataforma para desenvolvimento de APIs e microsserviços em Node.js

Important: This is a community translation into Portuguese. For the latest information, see the English version.

Important: LoopBack 4 é o próximo passo na evolução do Loopback. Ainda está em desenvolvimento inicial e ainda não foi lançado.

GitHub Repo

O código do framework LoopBack 4 está sendo desenvolvido em um único “mono-repositório”, loopback-next, ao invés de múltiplos repositórios, como na v3. Apesar disso, os exemplos e componentes desenvolvidos externamente estarão em repositório separados.

Construído para desenvolvedores de API

  • Defina seus schemas e endpoints da API usando o padrão OpenAPI.
  • Escreva seus endpoints com JavasScript moderno usando ES2017, async / await, módulos.
  • Use seus endpoints e schema como “source of truth” sem a necessidade de geração de código.

Construído para times

  • Revise as alterações nos endpoints da API sem procurar através do JavaScript.
  • Manter consistência automatizando a validação de seus endpoints e schemas.
  • Suporte de primeira classe para TypeScript (JavaScript fortemente tipado).

Construído para sua plataforma

  • Use LoopBack como ponto de partida para sua própria estrutura ou plataforma.
  • Construa bibliotecas de componentes reutilizáveis ​​de forma padronizada.
  • Integre com bancos de dados, serviços web e outras plataformas usando conectores.
